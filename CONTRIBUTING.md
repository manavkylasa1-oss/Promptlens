# Contributing to PromptLens

Thank you for wanting to make PromptLens better! Here's how to contribute.

---

## Ways to Contribute

### 1. Add Examples
The most valuable contribution. Add real before/after prompt transformations to the
`/examples` folder. Use the existing format:
- ❌ Before (the weak prompt)
- PromptLens Score with brief reasons
- ✅ Improved version
- What made the difference (3 bullets)

### 2. Improve the Core Framework
If you find a gap in the scoring dimensions, a missing scenario, or a better way to
explain something — open an issue first to discuss, then submit a pull request.

### 3. Translate PromptLens
Fork the repo and translate `promptlens.md` and `README.md` into another language.
Name the files `promptlens.[language-code].md` (e.g. `promptlens.es.md`).

### 4. Report Issues
Found a case where PromptLens gives bad advice? Open an issue with:
- The original prompt
- What PromptLens suggested
- What you think it should have suggested instead

---

## Pull Request Guidelines

- Keep PRs focused — one change per PR
- New examples must follow the existing format exactly
- Changes to core framework files (`promptlens.md`, `SKILL.md`) should be discussed
  in an issue first
- Write clearly — PromptLens is for everyone, not just experts

---

## Code of Conduct

Be respectful. PromptLens is a community project for helping people — keep that spirit.
